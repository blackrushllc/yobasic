In this chapter you’ll learn how to run your first BASIC programs in the YoBASIC web IDE.

Example 1 – Hello World

  1. Click the button to open this program in a new editor tab.
  2. Then click the Run button or press F9 to run the program.

HELLO.BAS

PRINT "Hello from YoBASIC!"
PRINT "This program ran from the tutorial sidebar."

This simple program prints two lines of text to the BASIC terminal. You can modify the text inside the quotes and run it again to see your changes.

Example 2 – Variables and Input

  1. Click the button to open this program in a new editor tab.
  2. Then click the Run button or press F9 to run the program.

VARS.BAS

REM Ask the user for their name and age, then respond
INPUT "What is your name? ", name$
INPUT "How old are you? ", age
PRINT "Nice to meet you, ", name$, "!"
PRINT "Next year you will be ", age + 1

This program demonstrates string variables, numeric variables, and reading user input using INPUT. It then prints a friendly message.

Example 3 – Decision Making with IF

  1. Click the button to open this program in a new editor tab.
  2. Then click the Run button or press F9 to run the program.

EVENODD.BAS

REM Determine if a number is even or odd
INPUT "Enter an integer: ", n
IF n MOD 2 = 0 THEN
  PRINT n, " is even"
ELSE
  PRINT n, " is odd"
END IF

This example uses IF/ELSE to branch logic and the MOD operator to check for even or odd numbers.

Example 4 – Loops with FOR/NEXT

  1. Click the button to open this program in a new editor tab.
  2. Then click the Run button or press F9 to run the program.

SQUARES.BAS

REM Print the squares of numbers 1 through 10
FOR i = 1 TO 10
  PRINT i, " squared is ", i * i
NEXT i

This short program introduces loops using FOR and NEXT by printing a small table of square numbers.
