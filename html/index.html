Quick-Start - Keyboard Shortcuts

  • Press F6 to toggle between your editor and the BASIC terminal.
  • Press F9 to run your program when the editor has focus.
  • Press F8 to run a selection within your program.
  • Press ESC to close side panels.
  • Click [Tron/Troff] to toggle Debug (Trace) Mode
  • Click [Clear] to clear the terminal
  • Click [Run (f9)] to run your code
  • Click [Exec (f8)] to run your code selection
  • Click [Clear] to clear the terminal
  • Click [Settings] to adjust preferences and colors
  • Click [Build] to publish a project
  • Click [Login] to create an identity, access shared files, and publish projects
  • Use the File Menu to Save and Load programs or start a project
